AI-Based Plankton Detection and Length Estimation

A tool that automates marine research processes by detecting and estimating the length of Calanus (plankton) in images through the use of deep learning and computer vision technology.
The web application has a simple user interface which allows users to provide a millimeter-to-pixel ratio (either by typing it as input or using our functionality to calculate it) and upload images for analysis.
With the ratio calculation functionality, users can upload an image with a reference object, enter the known length of the object in millimeters, place a point at the start and end of the object, and let our program calculate the ratio which will be used to convert lengths in pixels to lengths in millimeters.
The application produces a dataset (text file) for each image with IDs, estimated length in millimeters, and detection confidence, along with an image where each detected Calanus is marked with a circle that represents its estimated length and a matching ID. Users can download a zip with the results.
A crucial part of the project involved training a custom YOLOv5s model to recognise Calanus. The resulting model is able to detect Calanus of varying sizes with a very high confidence. It has also been trained to not include the tails in the bounding box since the client wanted the tails excluded from the length estimation.
Accurately estimating the length was a major challenge. Through trial and error, we found that the best approach was to binarize the image, find the largests contours within the bounding boxes of the detected Calanus, find the minimum enclosing circle of each contour (boxes are not good when dealing with rotated objects), and use the diameter as a representation of the length.
❮ ❯
Made with: Python, JavaScript, Flask, YOLOv5, OpenCV

Background:

Scientists are currently developing a sustainable harvest of Calanus, an underexploited species of zooplankton which is considered an ultra-efficient food source for fish and shrimp in marine farming. As part of this development, samples of Calanus catch from different trawls are being analysed to determine the size selectivity and catch efficiency of the gear. This involves spending several days manually identifying and measuring the length of Calanus in many images, since no existing solutions are able to both detect and measure Calanus automatically.

Solution:

This project aims to close this gap by offering an AI-based solution to Calanus detection and length estimation which lets users input one or more images and a millimeter-to-pixel ratio and get a list of all the lengths in millimeters in a text file as output. The tool has been developed by training a YOLOv5 model to recognise Calanus and using a variety of computer vision methods to estimate the length of the detected Calanus. 95.8% precision and 92% recall rates for Calanus detection have been demonstrated, as well as an error rate of less than 10% for more than 95% of the length estimations. The Calanus detection and length estimation functionality has been deployed in a user-friendly Flask web application.

My Rent Buddy: A Full-Stack Car Rental Website

A dynamic multi-page website where users will have access to different content and functionality depending on what type of user is logged into the system (renter/admin).
A registration form is available which lets new users join the system. Input like email, phone number (Australian), and password is validated through the use of regular expressions.
The renter user type can rent a car, return a car, search for cars, view current rentals, view previous rentals, and view all available cars.
The admin user type can view all cars in the system, view all available cars, view all rented cars, insert a car into the system, search for cars, and change the status of a car.

Made with: PHP, MySQLi

Background:

This project was built as part of a university subject on server-side programming. The purpose of the project was to create an interactive full-stack website for managing car rentals by using object-oriented PHP and a MySQL database.
